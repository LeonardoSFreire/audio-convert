Endpoints Disponíveis

Método Endpoint Descrição
GET /api/tags Listar modelos instalados
POST /api/generate Gerar texto/completion
POST /api/chat Chat conversacional com histórico
POST /api/pull Baixar novo modelo
DELETE /api/delete Remover modelo

Exemplos de Uso

1. Listar Modelos

curl 
curl http://207.244.247.167:11434/api/tags

2. Gerar Texto

curl 
curl -X POST http://207.244.247.167:11434/api/generate \
  -H "Content-Type: application/json" \
  -d '{
    "model": "llama3.1:8b",
    "prompt": "Explique Docker em 3 linhas",
    "stream": false
  }'

3. Chat com Contexto

curl 
curl -X POST http://207.244.247.167:11434/api/chat \
  -H "Content-Type: application/json" \
  -d '{
    "model": "llama3.1:8b",
    "messages": [
      {"role": "user", "content": "Qual a capital da França?"}
    ],
    "stream": false
  }'

4. Baixar Novo Modelo

curl 
curl -X POST http://207.244.247.167:11434/api/pull \
  -d '{"name": "qwen2.5:7b"}'
